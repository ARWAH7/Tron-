# 波场 (TRON) 哈希走势实时分析大盘 (路图增强版)

这是一个专为波场区块链设计的专业级哈希数据可视化工具。它通过连接 TronGrid API，实时捕获区块哈希，并自动生成博弈级的可视化路图（大路、珠盘路），帮助用户深度分析哈希末位数字的“单双”与“大小”走势。

## 核心功能

- **全方位可视化路图**：
  - **大路 (Big Road)**：经典趋势追踪，连续结果自动成列。
  - **珠盘路 (Bead Road)**：按出块顺序精确填充的 6xN 矩阵，还原历史全貌。
- **2x2 深度分析面板**：
  - **单双模块**：红蓝经典配色，实时统计单双比例。
  - **大小模块**：橙紫专业配色，实时统计大小分布。
- **高性能实时监听**：
  - **3秒极速同步**：毫秒级监听波场主网出块，实时推送。
  - **内存级缓存**：支持数千个区块的流畅滚动与快速检索。
- **高级采样逻辑**：
  - **自定义间隔**：支持 1, 20, 60, 100 等任意区块采样步长。
  - **偏移对齐**：支持锁定起始区块高度，实现特定规则的对齐分析。

## 运行配置

### 1. 环境要求
- 现代浏览器 (Chrome, Safari, Edge, Firefox)。
- TronGrid API Key (必填，用于获取主网实时数据)。

### 2. 获取 API Key
- 前往 [TronGrid.io](https://www.trongrid.io/) 免费注册并创建 API KEY。

## 本地部署

1. **获取代码**：下载本项目的 HTML、JS 与资源文件。
2. **启动服务**：
   ```bash
   # 使用 Python 快速启动
   python -m http.server 8000
   ```
3. **访问地址**：浏览器打开 `http://localhost:8000`。
4. **鉴权配置**：点击界面右上角“设置”图标，粘贴您的 TronGrid API Key 即可启动监听。

## 分析逻辑说明

- **单双 (Parity)**：
  - **单 (ODD)**：哈希末位数字为 1, 3, 5, 7, 9。
  - **双 (EVEN)**：哈希末位数字为 0, 2, 4, 6, 8。
- **大小 (Size)**：
  - **大 (BIG)**：哈希末位数字 5 - 9。
  - **小 (SMALL)**：哈希末位数字 0 - 4。

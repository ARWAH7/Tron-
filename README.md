
# 波场 (TRON) 哈希走势实时分析大盘

这是一个专为波场区块链设计的实时哈希数据可视化工具。它通过连接 TronGrid API，实时捕获区块哈希，并自动分析哈希末尾数字的“单双”与“大小”走势，为数据分析提供直观的参考。

## 项目特点

- **全中文界面**：专为中文用户优化，术语准确、布局清晰。
- **3秒极速响应**：数据刷新频率设定为3秒（波场出块时间约为3秒），确保数据同步。
- **无感启动**：配置 API Key 后直接从最新区块开始监听，无需等待冗长的历史数据拉取。
- **瞬时切换无丢失**：采用“全量数据集”架构，在不同采样间隔（单区块/20/60/100）之间切换时，数据瞬时展示且绝不丢失。
- **多采样周期**：支持单区块、20区块、60区块、100区块四种采样模式。
- **高性能渲染**：采用 React 19 并结合 Memo 化组件及内存缓存技术，即使长时间挂机运行也不会卡顿。

## 运行配置

### 1. 环境要求
- 现代浏览器（Chrome, Edge, Safari, Firefox）。
- 稳定的网络连接（需能访问 `api.trongrid.io`）。

### 2. 获取 API Key
- 访问 [TronGrid 官网](https://www.trongrid.io/)。
- 注册账号并创建一个 API Key。
- 建议使用付费或高配 API Key 以获得更好的每秒请求限额。

## 本地运行步骤

由于本项目采用现代 ES Modules (ESM) 架构，无需复杂的打包环境即可运行：

1. **获取代码**：
   将项目所有文件存放在同一个目录下。

2. **启动本地服务**：
   使用任何轻量级静态服务器运行。例如使用 Node.js 的 `serve` 或 Python 的内置服务：
   ```bash
   # 使用 Python 启动
   python -m http.server 8000
   
   # 或者使用 Node.js 的 npx
   npx serve .
   ```

3. **访问应用**：
   打开浏览器，访问 `http://localhost:8000` (或服务器指定的端口)。

4. **配置应用**：
   - 点击右上角的“设置”图标（齿轮形状）。
   - 在弹出框中输入您的 TronGrid API Key。
   - 点击“保存并开始监听”，系统将立即加载当前最新区块并开始每3秒一次的实时监听。

## 走势逻辑说明
- **单双 (Parity)**：区块哈希最后一个数字为 1, 3, 5, 7, 9 时为“单”，为 0, 2, 4, 6, 8 时为“双”。
- **大小 (Size)**：区块哈希最后一个数字在 0-4 之间为“小”，在 5-9 之间为“大”。
